Creating roles interface in Craftable

How to generate the roles interface for your craftable admin panel

Step 1

First generate role interface in the admin area by typing the command below

php artisan admin:generate roles

Then type yes when asked Do you want to attach generated permissions to the default role now? (yes/no) [yes]:


Step 2

Run the command npm run dev


Step 3

Navigate to your admin panel and refresh. the roles created will appear under the content section. look at image below


Step 4

Create a new role by selecting the new role button. (for my example i used editor as the role)
